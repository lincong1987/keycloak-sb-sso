#特效组件说明


rotate-star 旋转星空
```vue
<motion-rotate-star :num="300"></motion-rotate-star>
```


star-line 星空连线
```vue
<motion-star-line :num="300"></motion-star-line>
```


point-wave 点阵波
```vue
<motion-point-wave :speed="5"></motion-point-wave>
```

victor 科技块
```vue
<motion-victor opacity="0.5" theme="red"></motion-victor>
```
